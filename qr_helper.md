# 二维码访问方式

本项目可以通过局域网二维码进行现场测试，也可以部署到 Render 后生成公网二维码用于正式活动。

## 方式一：局域网二维码

1. 电脑和手机连接同一个 WiFi。
2. 电脑运行 `python app.py`。
3. Windows 使用 `ipconfig` 查看 IPv4 地址。
4. 假设电脑 IP 是 `192.168.1.8`，则手机访问：

```text
http://192.168.1.8:5000
```

5. 将这个链接复制到二维码生成工具中，生成二维码。
6. 注意：电脑不能关机，项目必须一直运行，手机和电脑必须在同一个 WiFi 下。

## 方式二：公网二维码

1. 将项目部署到 Render 或其他云平台。
2. 获得公网网址。
3. 将公网网址生成二维码。
4. 该二维码可放在宣传单、政策明白卡、帆布包、拓印卡、活动展板和公众号推文中。

推荐正式活动使用公网二维码。局域网二维码更适合开发测试、课堂演示或临时现场调试。
