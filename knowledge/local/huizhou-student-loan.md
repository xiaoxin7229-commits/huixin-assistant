# 基本介绍

本条目用于回答惠州户籍学生如何确认生源地助学贷款办理渠道。

# 适用对象

户籍或生源地在惠州、需要申请首贷或续贷的学生及共同借款人。

# 核心内容

## 政策事实

PDF 描述了线上申请、首贷核验和续贷等常见环节。

## 经验建议

先通过国家开发银行学生在线系统和户籍所在地县级资助中心确认当年安排。

## 需要当地确认

额度、利率、期限、现场或远程办理方式、材料和结算账户均不得仅依据 PDF 确定。

# 办理流程

查看当年公告 → 登录官方系统 → 填写申请 → 按首贷或续贷要求核验 → 签订合同 → 向学校提交回执信息。

# 常见问题

是否必须到现场？首贷和续贷要求可能不同，以系统及户籍所在地县级资助中心通知为准。

# 注意事项

本条目不提供未经核验的电话、地址或办理时间；不要通过搜索广告进入贷款系统。

用户提供的《惠州本地知识库.pdf》。需要惠州市或县区学生资助部门进一步核验。
