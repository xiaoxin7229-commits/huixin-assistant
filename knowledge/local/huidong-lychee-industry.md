# 基本介绍

本条目整理 PDF 中关于惠东荔枝产业、产销对接、物流和文旅融合的场景。

# 适用对象

惠东荔枝种植户、合作社、返乡青年和村级服务人员。

# 核心内容

## 政策事实

PDF 提到重点产区、加工项目、采摘路线和村级物流合作等内容，但未逐项附官方依据。

## 经验建议

产销活动应先确认产品质量、订单责任、包装冷链成本和平台结算规则，再开展宣传或团购。

## 需要当地确认

种植面积、产量、项目主体、收寄点数量、折扣和降本比例均须向农业农村、文旅或物流主体核实。

# 办理流程

明确产品与产地 → 联系合作社或村集体 → 核对采购和物流条件 → 签订可追溯订单 → 留存结算记录。

# 常见问题

村委是否一定提供统一物流优惠？不一定，应以所在村当期合作安排为准。

# 注意事项

不得以未经核实的产量、收益或折扣作宣传承诺；农产品质量责任不能因线上销售而免除。
