# 基本介绍

本条目整理 PDF 中关于惠东全域土地综合整治和海丝文化遗产保护的问答场景。

# 适用对象

涉及土地项目、村庄建设、文旅活动或疑似文物发现的村民和项目参与者。

# 核心内容

## 政策事实

发现疑似文物或遗迹时应保护现场并向主管部门报告，不应擅自挖掘、搬运或买卖。

## 经验建议

土地整治或建设项目应先查明审批边界、权属和文物保护要求，再开展施工。

## 需要当地确认

PDF 中的投资规模、覆盖范围、工程名称、实施年份和遗产名录状态未附直接文件，必须逐项核验。

# 办理流程

确认事项类型 → 向自然资源或文物主管部门咨询 → 提交位置和现场信息 → 等待专业核查 → 按正式意见处理。

# 常见问题

发现旧瓷片能否自己保存？不应自行判断，应保护现场并向当地文物主管部门报告。

# 注意事项

本条目不提供未经确认的电话和办理地点；涉及土地权利或执法问题应以书面官方意见为准。
