# 基本介绍

本条目帮助助农经营者识别虚假宣传、伪造证明和“悲情营销”等风险。

# 适用对象

参与农产品宣传、直播、团购、代销或品牌建设的广东和惠州经营者。

# 核心内容

## 政策事实

不得伪造产地、质量检测、认证或困难故事，不得发布虚假或引人误解的广告。

## 经验建议

用可核验的生产记录、检测材料、发货数据和售后规则建立信任，不承诺无法证明的功效、销量和收益。

## 需要当地确认

具体标签、许可、检测和广告表述应向市场监管、农业农村部门或专业人员咨询。

# 办理流程

盘点宣传材料 → 核对证明真实性 → 删除夸大表述 → 建立溯源和投诉处理记录 → 定期复查。

# 常见问题

可以借用其他基地的检测报告吗？不可以。证明材料必须对应实际产品和批次。

# 注意事项

适用地区为广东/惠州。农业病虫害、防治方案不能代替农技人员诊断；禁止直接推荐具体农药剂量。
