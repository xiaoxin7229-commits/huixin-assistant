# 基本介绍

本条目处理“线上订单来了以后如何稳定交付”的问题。

# 适用对象

开展生鲜农产品电商、团购或直播销售的农户、合作社和乡村青年。

# 核心内容

## 政策事实

生鲜交付应符合适用的食品安全、包装标识、运输和消费者权益要求。

## 经验建议

在开售前确定采收能力、分级规则、包装材料、冷链时效、破损赔付和异常天气预案。

## 需要当地确认

是否需要冷链、可承诺时效、村级集运点和具体物流价格应向产品技术人员及承运方确认。

# 办理流程

核定可售量 → 确认包装与温控 → 约定承运责任 → 按批次出库 → 跟踪签收 → 处理售后并复盘损耗。

# 常见问题

有订单后再找物流可以吗？风险较高，生鲜产品应在开售前完成承运和包装测试。

# 注意事项

适用地区为广东/惠州。农业病虫害、防治方案不能代替农技人员诊断；本条目不涉及农药使用剂量。
