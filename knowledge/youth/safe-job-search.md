# 基本介绍

本条目帮助青年选择正规招聘渠道并识别常见求职骗局。

# 适用对象

高校毕业生、乡村青年及首次进入劳动力市场的求职者。

# 核心内容

## 政策事实

招聘活动不得扣押身份证等证件，不应以招聘名义收取不合理费用或诱导求职者办理培训贷款。

## 经验建议

优先使用人社部门、学校和可信招聘平台；核对企业登记、岗位地点、劳动合同和薪酬构成。

## 需要当地确认

投诉举报和法律援助渠道应按事件发生地向人社、市场监管、公安或法律服务机构确认。

# 办理流程

核验单位 → 询问岗位细节 → 保存沟通记录 → 审阅合同 → 拒绝押金和证件扣押 → 发现风险及时退出并求助。

# 常见问题

入职前要求贷款培训是否正常？风险很高，应暂停办理并向学校或人社部门核实。

# 注意事项

不要提供银行卡验证码，不参与要求发展下线的活动；遇到人身威胁应立即报警。
