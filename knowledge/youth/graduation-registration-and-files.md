# 基本介绍

本条目帮助毕业生处理毕业去向登记和档案转递，不提供个案档案归属结论。

# 适用对象

高校应届毕业生、暂未就业毕业生和跨地区就业青年。

# 核心内容

## 政策事实

学生档案通常由学校按规定转递至有档案管理权限的单位或公共就业人才服务机构，个人不应自行长期保管。

## 经验建议

离校前核对毕业去向登记、档案接收单位名称、地址和查询方式，并保留转递记录。

## 需要当地确认

系统入口、材料、接收机构和办理时间应由学校及拟接收机构确认。

# 办理流程

确认毕业去向 → 咨询学校就业部门 → 联系接收机构 → 填写准确转递信息 → 离校后查询到档。

# 常见问题

档案能否自己携带？通常不应自行携带，应由学校按规定渠道转递；特殊情况须听从学校和接收机构说明。

# 注意事项

不要轻信收费“档案激活”服务；转递信息错误应尽快联系学校更正。
