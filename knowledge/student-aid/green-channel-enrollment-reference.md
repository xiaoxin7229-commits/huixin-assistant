# 基本介绍

本条目整理家庭经济困难新生在报到阶段咨询“绿色通道”的场景。

# 适用对象

入学前暂时无法筹足学费或住宿费、需要向录取高校申请帮助的新生及家长。

# 核心内容

## 政策事实

PDF 将绿色通道描述为帮助困难新生先办理入学、再衔接困难认定和其他资助的机制。

## 经验建议

收到录取材料后尽早联系学校招生或资助部门，确认报到入口和所需说明材料。

## 需要当地确认

具体材料、办理窗口、线上入口和后续资助安排由学校确定。

# 办理流程

查阅录取材料 → 联系学校确认 → 按要求提交申请 → 报到时完成核验 → 入学后衔接困难认定及其他资助。

# 常见问题

绿色通道是否等于免除学费？不是。它主要解决报到衔接问题，后续费用和资助方式以学校审核为准。

# 注意事项

不要因暂时缴费困难放弃报到；也不要相信收费代办“绿色通道”的个人。
