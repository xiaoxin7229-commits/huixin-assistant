# 基本介绍

涉及资格、金额、材料、办理时间、审核结果和贷款合同的问题，应以官方渠道答复为准。

# 适用对象

需要核实国家学生资助、广东省学生资助或国家开发银行助学贷款信息的学生和家长。

# 核心内容

- 全国学生资助管理中心官网：https://www.xszz.edu.cn/
- 国家开发银行助学贷款学生在线系统：https://www.csls.cdb.com.cn/
- 国家开发银行助学贷款服务热线：95593。
- 广东省教育厅网站：https://edu.gd.gov.cn/
- 学生所在学校学生资助管理部门或辅导员。
- 户籍所在地县级教育部门、学生资助管理中心；具体电话和地址以当地政府网站当年公告为准。

# 办理流程

1. 先确定咨询事项属于学校资助、地方资助还是助学贷款。
2. 优先查看对应官网或学生在线系统。
3. 需要个案确认时，联系学校或当地学生资助管理部门。
4. 涉及贷款账户和合同问题时，通过国家开发银行官方系统或95593咨询。

# 常见问题

## 惠芒Go能直接查我的审核结果吗？

不能。惠芒Go不连接个人业务系统，也不作资格认定。请登录官方系统或联系办理部门。

## 网上搜索到的电话号码都可靠吗？

不一定。地方电话可能变化，应从当地政府、教育部门或学校官网核对。

# 注意事项

- 惠芒Go不收取办理费用，不索要验证码和密码。
- 本项目不是政府办理平台。
- 热线服务时间可能调整，请以国家开发银行学生在线系统当前页面为准。
