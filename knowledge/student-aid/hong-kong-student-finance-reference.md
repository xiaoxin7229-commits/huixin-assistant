# 基本介绍

本条目整理 PDF 中关于香港专上学生资助计划、入息审查和贷款计划的辨别场景。

# 适用对象

在香港修读专上课程、需要判断课程和身份适用计划的学生及家庭。

# 核心内容

## 政策事实

PDF 提到不同课程资助性质、修读模式和经济审查选择可能对应不同计划。

## 经验建议

先确认课程是否公帑资助、是否全日制及申请人年龄，再通过香港学生资助处官方系统核对。

## 需要当地确认

计划名称、年龄限制、利率、复核期限、材料及电子办理入口均具有时效性，PDF 未给出可核验的直接来源页面。

# 办理流程

确认课程属性 → 在官方系统选择计划 → 核对审查方式 → 准备收入和资产证明 → 提交 → 保存结果及复核期限。

# 常见问题

资助不足以覆盖学费怎么办？应向香港学生资助处或院校核对可组合的资助与贷款安排。

# 注意事项

不要将内地资助规则直接套用于香港计划，也不要根据本草稿判断资格或利息。
