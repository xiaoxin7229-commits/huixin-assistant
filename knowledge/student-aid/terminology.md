# 基本介绍

本文件用通俗语言解释学生资助咨询中的常见术语，不替代正式政策定义。

# 适用对象

对奖助学金、贷款、困难认定和入学资助术语不熟悉的学生、家长及社区群众。

# 核心内容

- **国家奖学金**：奖励特别优秀学生的国家奖学项目。
- **国家励志奖学金**：奖励资助品学兼优且家庭经济困难学生的项目。
- **国家助学金**：主要帮助家庭经济困难学生解决学习生活方面的基本需要。
- **国家助学贷款**：帮助家庭经济困难学生支付学费、住宿费并补充部分生活费的政策性贷款，需要按合同履行还款义务。
- **生源地信用助学贷款**：通常在学生入学前户籍所在地申请的国家助学贷款。
- **校园地国家助学贷款**：通常在入学后通过高校申请的国家助学贷款。
- **共同借款人**：按生源地贷款规定与学生共同承担合同义务的人，具体条件由官方规定和审核确定。
- **首贷**：学生第一次申请相应助学贷款。
- **续贷**：已经获得过贷款的学生再次申请下一学年贷款。
- **绿色通道**：高校允许暂时无法缴纳学费的家庭经济困难新生先办理入学手续，再衔接困难认定和资助项目的制度安排。
- **家庭经济困难认定**：学校按规定了解和评估学生家庭经济状况，为相关资助工作提供依据的过程。
- **贴息**：按政策由财政承担符合规定期间的贷款利息。
- **回执**：生源地贷款办理后，高校按要求确认学生入学等信息的相关手续。

# 办理流程

术语本身没有独立办理流程。遇到不理解的词，应结合具体项目通知、贷款合同或学校说明理解。

# 常见问题

## 助学金和助学贷款一样吗？

不一样。助学金通常不需要偿还；助学贷款属于贷款，需要按合同还款。

## 奖学金和助学金可以混为一谈吗？

不能。不同项目的目标、对象和评审条件不同，应分别查看通知。

# 注意事项

同一术语在具体文件中的适用范围可能更严格，应以正式文件为准。
