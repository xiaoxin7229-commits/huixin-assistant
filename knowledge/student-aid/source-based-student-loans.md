# 基本介绍

生源地信用助学贷款是国家助学贷款的一种。符合条件的家庭经济困难学生，通常在入学前户籍所在地的县级学生资助管理部门申请，用于帮助解决在校期间学费、住宿费和部分日常生活费。

# 适用对象

主要面向符合国家和当地办理条件的全日制普通本专科学生、高职学生、预科生、第二学士学位学生和研究生。是否符合条件，以当地学生资助管理部门审核为准。

# 核心内容

- 申请应通过国家开发银行助学贷款学生在线系统、当地学生资助管理部门等官方渠道办理。
- 申请通常涉及学生和共同借款人。共同借款人的具体资格不得仅凭通用说明判断。
- 自2024年秋季学期起，全日制普通本专科学生每人每年申请贷款额度最高不超过20000元，全日制研究生最高不超过25000元。
- 贷款优先用于学费和住宿费，超出部分可用于弥补日常生活费。
- 贷款额度、利率、期限和还款安排以现行政策、贷款合同和官方系统为准。

# 办理流程

1. 查看户籍所在地学生资助管理部门当年通知。
2. 通过国家开发银行助学贷款学生在线系统注册或登录。
3. 按通知填写申请并准备材料。
4. 按当地要求完成线上或现场审核、合同签订。
5. 入学后按学校要求提交受理证明或完成回执确认。

首次申请和续贷的办理方式可能不同；不得根据本文件推断具体受理日期、地点或材料清单。

# 常见问题

## 可以只在网上完成吗？

不同地区、不同申请类型的要求可能不同。请查看当地当年通知，确认是否需要现场办理。

## 共同借款人一定是父母吗？

通常优先考虑父母或法定监护人，但其他情形的条件由当地规定和正式审核确定。

# 注意事项

- 不要向非官方人员提供身份证号、银行卡号、验证码或系统密码。
- 不要相信收费代办、提前放款或“内部名额”等说法。
- 本文件不代替贷款合同解释和资格审核。
