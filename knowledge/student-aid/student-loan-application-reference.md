# 基本介绍

本条目整理 PDF 中关于生源地信用助学贷款、首贷和续贷的场景，不替代经办银行或县级学生资助管理部门的说明。

# 适用对象

因家庭经济困难需要了解国家助学贷款的学生及共同借款人。

# 核心内容

## 政策事实

PDF 说明贷款主要用于帮助学生解决学费、住宿费等就学支出。贷款额度、利率、贴息、期限和结算账户均属于高风险时效信息。

## 经验建议

办理前只使用教育部门、国家开发银行或当地资助中心确认的入口，提前核对身份证明、录取或在读信息及共同借款要求。

## 需要当地确认

PDF 中的额度、利率公式、宽限期、APP 操作及材料清单均须以办理年度官方公告为准。

# 办理流程

确认贷款类型 → 查询当年通知 → 在线申请或预约 → 按要求核验身份与材料 → 签订合同 → 保存回执和还款信息。

# 常见问题

续贷是否一定需要现场办理？各地区和年度安排不同，应以系统提示及当地资助中心通知为准。

# 注意事项

不向陌生链接提交身份证、银行卡或验证码；合同与还款责任应由借款人认真阅读确认。
