# 基本介绍

广东省学生资助体系覆盖学前教育、义务教育、普通高中、中等职业教育和普通高等教育，通过奖、贷、助、补、免、勤等方式帮助家庭经济困难学生，并奖励品学兼优学生。

# 适用对象

在广东省接受相应阶段教育、符合国家或广东省具体项目条件的学生。各项目的户籍、学籍、家庭经济情况和年级要求不同，应分别核对。

# 核心内容

- 学前及义务教育阶段以保障基本受教育权利和困难群体支持为主。
- 普通高中和中等职业教育阶段包括助学金、免学费或免学杂费、奖学金等项目。
- 高等教育阶段包括国家奖助学金、国家助学贷款、绿色通道、勤工助学、困难补助及广东省相关专项资助。
- 具体项目、标准、范围和申请时间可能调整，历史文件中的金额不能直接视为当前标准。

# 办理流程

1. 明确学生学段、学校、户籍和困难情况。
2. 查看学校、当地教育部门或学生资助管理部门最新通知。
3. 按具体项目提交申请和必要材料。
4. 配合学校或主管部门审核、公示和发放程序。

# 常见问题

## 广东学生只能申请广东省项目吗？

不是。符合条件的学生可按规定申请国家统一资助项目，也可能适用广东省或学校项目，具体以项目通知为准。

## 历史文件里的金额现在还能直接使用吗？

不能。政策标准可能调整，必须以申请年度的最新官方文件和学校通知为准。

# 注意事项

本文件只说明政策体系，不作资格认定。涉及广东省专项资助时，需要官方进一步核验具体年度、地区和学校要求。
