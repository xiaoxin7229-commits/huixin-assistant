# 基本介绍

本条目用于社区志愿服务报名、信息保护和活动安全咨询。

# 适用对象

拟参加社区志愿活动的学生、青年、村民及组织者。

# 核心内容

## 政策事实

志愿服务应遵循自愿、无偿、平等和诚信原则；组织方应根据活动风险进行培训和安全保障。

## 经验建议

通过正式志愿服务平台或社区组织报名，确认工作内容、时间、联系人、风险和保险安排。

## 需要当地确认

注册系统、备案材料、保险范围及特殊岗位资质由当地规定和活动组织方确定。

# 办理流程

查看正规招募 → 核对组织者 → 提交必要信息 → 接受培训 → 参加活动 → 记录服务并反馈问题。

# 常见问题

志愿服务可以收取报酬吗？志愿服务本身强调无偿；合理补贴或保障应按活动规则公开说明，不得借志愿名义营利。

# 注意事项

不得泄露服务对象隐私；高风险活动应有专业指导和保险，不得强行指派不适合的任务。
