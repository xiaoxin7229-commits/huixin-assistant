# 基本介绍

本条目帮助家庭了解如何核查社区托育点的资质、安全和服务安排。

# 适用对象

需要婴幼儿照护服务的家庭、社区工作人员和托育项目参与者。

# 核心内容

## 政策事实

托育服务涉及场地、消防、卫生健康、人员资质和婴幼儿安全，不能仅凭“社区项目”名称判断合规。

## 经验建议

家长应核对备案或许可信息、照护人员、接送制度、应急预案、收费和退费规则。

## 需要当地确认

PDF 所述建设模式和服务标准未对应具体地区文件，是否设点及如何收费须向当地卫生健康等部门确认。

# 办理流程

查询本地服务名单 → 实地查看 → 核验资质和人员 → 阅读合同 → 试托并持续沟通。

# 常见问题

社区提供场地是否代表机构一定合规？不代表，仍需核验运营主体和相关备案、消防及安全要求。

# 注意事项

不得以远程监控替代现场照护和专业管理；不要公开儿童影像及健康信息。

# 来源

用户提供的《社区服务知识库.pdf》。需要当地卫生健康、消防和市场监管部门进一步核验。
