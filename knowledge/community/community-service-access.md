# 基本介绍

本条目处理居民不知道“某项事情去哪里问、线上不会办怎么办”的场景。

# 适用对象

普通居民、村民、老年人及需要社区协助的家庭。

# 核心内容

## 政策事实

不同事项由不同部门负责，社区通常承担咨询、协助或转介功能，不一定具有最终审批权。

## 经验建议

先说明事项、户籍或居住地、紧急程度和已有材料，请社区帮助确认主管部门及线上、线下渠道。

## 需要当地确认

服务清单、窗口地址、收费、材料和办理时限必须以当地正式公布为准。

# 办理流程

明确需求 → 咨询社区综合窗口 → 确认主管部门 → 准备材料 → 办理并保存受理凭证。

# 常见问题

不会使用智能手机怎么办？可询问当地是否保留线下窗口、人工帮办或无障碍服务。

# 注意事项

不要向非工作人员透露验证码或完整金融账户信息；紧急医疗、消防或人身安全事项应直接联系相应紧急渠道。
