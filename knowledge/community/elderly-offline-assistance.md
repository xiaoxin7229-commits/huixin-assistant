# 基本介绍

本条目整理 PDF 中关于不会使用智能手机的老年人如何寻求线下或人工协助的场景。

# 适用对象

不熟悉线上操作的老年居民及协助其办事的家属。

# 核心内容

## 政策事实

具体事项是否提供线下窗口、人工帮办或上门服务，需要由当地社区和主管部门确认。

## 经验建议

先联系社区说明事项和困难，询问最近的线下窗口、所需材料及是否可由家属陪同或代办。

## 需要当地确认

服务地址、时间、代办授权和收费标准必须以当地公布为准。

# 办理流程

说明需求 → 询问线下渠道 → 准备必要材料 → 由本人或合规代理人办理 → 保存受理凭证。

# 常见问题

社区能否代替本人完成所有事项？不能一概而论，涉及身份核验或个人意愿的事项可能要求本人办理。

# 注意事项

协助者不得索取验证码或擅自操作资金账户；身份证件使用后应及时收回。
